# cashew
My very own Redis inspired caching service

This is my take of [Build Your Own Redis Server](https://codingchallenges.fyi/challenges/challenge-redis)

## Roadmap

| Goal                                                                      | Status                |
| ------------------------------------------------------------------------- | --------------------- |

| De-serialise RESP                                                         | Pending               |
| A server that listens (`PING`, `ECHO`)                                    | Pending               |
| Set and Get keys                                                          | Pending               |
| Concurrent access                                                         | Pending               |
| Implement expiry options                                                  | Pending               |
| Implement `EXISTS`, `DEL`, `INCR`, `DECR`, `LPUSH`, `RPUSH`, `SAVE`       | Pending               |
